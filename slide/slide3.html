Problem Definition
Two-Stage End-to-End ABSA Pipeline: ATE + ASC
1
Aspect Term Extraction (ATE)
Input: Raw review sentence
Task: BIO sequence labeling (like NER) to extract aspect spans
Output: Aspect terms with BIO tags
"The [B-ASP]clam [I-ASP]chowder was incredible but the [B-ASP]waiter was rude."
2
Aspect Sentiment Classification (ASC)
Input: (sentence, extracted aspect term)
Task: Classify sentiment as Positive / Negative / Neutral
Output: Sentiment label for each aspect
("...clam chowder was incredible...", "clam chowder") → Positive

Key Insight: The pipeline chains ATE and ASC to extract fine-grained sentiment from raw text
ATE → Extracts aspect terms using BIO labeling
ASC → Classifies sentiment for each extracted aspect
